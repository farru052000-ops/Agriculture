About OptiCrop

OptiCrop is an AI-powered crop recommendation system developed to help farmers select the most suitable crop based on soil nutrients and environmental conditions. By using Machine Learning, the system predicts the best crop that can maximize agricultural productivity and improve farming decisions.

Project Objectives

  • Recommend the most suitable crop for cultivation.
  • Improve agricultural productivity using Machine Learning.
  • Reduce crop selection errors.
  • Support farmers with data-driven decisions.

Input Parameters

  • Nitrogen (N)
  • Phosphorus (P)
  • Potassium (K)
  • Temperature
  • Humidity
  • Soil pH
  • Rainfall

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-learn
  • HTML
  • CSS

Machine Learning Models

  • Logistic Regression
  • K-Nearest Neighbors (KNN)
  • Decision Tree
  • Random Forest
  • Naive Bayes (Best Model)
